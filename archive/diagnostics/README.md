# Diagnostic Reports Archive

This directory contains historical diagnostic reports for troubleshooting issues.

## Contents

- **COMPLETE_DIAGNOSTIC.md** - Complete diagnostic report for map display issues

## Purpose

Diagnostic reports are archived here after issues are resolved. They provide:
- Historical context for debugging similar issues
- Documentation of resolved problems
- Examples of troubleshooting methodology
- Configuration snapshots at specific points in time

## When to Add New Diagnostics

Add new diagnostic reports here when:
- A complex issue has been resolved
- The diagnostic process may help future troubleshooting
- The report contains valuable configuration snapshots
- The issue is no longer active but may recur

## Current Issues

For active troubleshooting, use:
- GitHub Issues
- docs/ directory for living documentation
- Root README.md for current status
